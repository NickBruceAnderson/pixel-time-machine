# Pixel Time Machine

One game demake per year, starting in 1972.

Built with Phaser 3 + Claude Code. Low-code driven, fully readable.

## The Project
Each folder is a year. Each year is a playable game in the browser.

- 1972 — Pong
- 1973 — (next)
- ...

## Stack
- Phaser 3 (JS)
- Claude Code
- GitHub Pages
- Wispr Flow
